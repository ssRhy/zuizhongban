# 八字分析系统

一个结合传统命理学与现代技术的个人命理分析系统，提供全面的八字分析和个性化建议。系统采用科学的算法将传统命理学知识数字化，为用户提供准确的分析结果和实用的日常指导。

## ✨ 主要功能

- 🎯 **八字分析**: 
  - 基于出生年月日时的命理分析
  - 天干地支五行属性计算
  - 八字强弱分析
  - 喜用神与忌神判定
  
- 🌈 **五行分析**: 
  - 五行能量平衡计算
  - 个性化平衡建议
  - 五行属性强弱评估
  - 日主特征分析

- 💎 **水晶推荐**: 
  - 基于八字特点的个性化推荐
  - 根据五行缺失进行补充
  - 每种水晶的具体功效说明
  - 多维度的选择标准

- 🎨 **每日运势**: 
  - 个性化幸运色推荐
  - 基于天干地支的颜色系统
  - 当日吉凶分析
  - RGB精确色值对照

- 📅 **活动建议**: 
  - 基于五行特点的日常活动建议
  - 个性化行动指南
  - 根据八字特征优化日程安排
  - 提供具体可行的建议

- 💫 **数字推荐**: 
  - 基于八字的幸运数字计算
  - 五行数字对应关系
  - 个性化数字选择

## 📦 安装部署

### 环境要求
- Python 3.x
- pip包管理器
- 现代浏览器（Chrome/Firefox/Safari）

### 依赖安装
```bash
pip install flask flask-cors requests
```

### 启动服务
```bash
python app.py
```
服务将在 http://127.0.0.1:5000 启动

## 📝 使用指南

### 1. 基础信息输入
- 访问系统首页
- 填写个人信息：
  - 姓名
  - 性别
  - 出生类型（农历/公历）
  - 出生年月日时分

### 2. 获取分析结果
系统将提供全面的分析报告，包括：
- 八字详解
- 五行分析
- 个性化建议
- 今日幸运色（带RGB值）
- 推荐水晶
- 建议活动
- 幸运数字

### 3. 实践建议
- 根据分析结果调整日常活动
- 参考幸运色选择穿着或环境装饰
- 选择推荐的水晶作为饰品或摆件
- 在重要决策中参考幸运数字
